# Overview

This is sample for

- react and react-redux
- routing

## Dependency

- php7
- composer
- nodejs >= 10
- yarn

## Installation

```
$ composer intall
$ yarn install
```

## run

```
$ cd public
$ php -S localhost:8000
```

and check following url

- localhost:8000/app/
- localhost:8000/app/auth/login
- localhost:8000/app/auth/signup
